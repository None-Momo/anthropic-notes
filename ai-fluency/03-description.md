# Description

## What you'll learn
- Write effective prompts/briefs that set context, goal, and constraints.
- Use structure, examples, and checklists to reduce ambiguity.

## Prompt structure
- **Goal**: what outcome and audience?
- **Context**: background, domain, tone, style.
- **Inputs**: data, examples, links.
- **Constraints**: length, format, must/forbidden items.
- **Validation**: how you'll judge success.

## Helpful techniques
- Role + task framing
- Provide exemplars (few-shot)
- Ask for outline before full draft
- Chain-of-thought or bullet reasoning
- Ask for self-check / rubric

## Template
```
You are <role>. Task: <goal>. Audience: <who>.
Context: <brief background>.
Constraints: <style/length/format/forbidden>.
Process: <outline first / show steps / cite sources>.
Quality bar: <how it will be evaluated>.
```

## Practice
- Rewrite one of your prompts using the template; compare outputs.
