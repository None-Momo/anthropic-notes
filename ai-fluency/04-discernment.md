# Discernment

## What you'll learn
- Review AI outputs for accuracy, safety, and fit-for-purpose.
- Catch common failure modes (hallucinations, gaps, bias).

## Review checklist
- **Accuracy**: facts cited, sources present? conflicting info?
- **Completeness**: requirements covered? edge cases?
- **Safety/ethics**: privacy, harm, bias, compliance.
- **Format**: correct structure, length, audience-ready.
- **Reasoning**: steps make sense? alternative answers considered?

## Tactics
- Ask for sources and verify a sample.
- Run a second pass: "critic" prompt on the output.
- Stress-test with counterexamples or adversarial cases.
- Compare against a short rubric (3–5 bullets).

## Practice
- Take an AI answer you have; run the checklist and revise.
